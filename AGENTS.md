# AGENTS.md

## Project overview
This repository contains the marketing website for RichLand Ltd., a Shunde-based fan manufacturer.
The current site is a static website focused on clean product presentation, OEM/ODM positioning, and inquiry conversion.

## Primary goal
Improve the website visually without overengineering it.
Prefer calm, modern, balanced layouts over flashy or experimental designs.

## Tech stack
- Static website
- Main file: `index.html`
- Product images currently live in the repository and should gradually be organized under `assets/`

## Design priorities
1. Clean and premium product presentation
2. Strong visual balance and spacing
3. Consistent image sizing and card layout
4. Clear product grouping:
   - Pedestal Fans
   - Table Fans
   - Wall Fans
   - Air Circulator & Floor Fans
   - Other Appliances (optional later)
5. Keep copy concise and professional
6. Preserve the B12-style brand positioning, but make the UI look more custom and polished

## Layout rules
- Hero section should use a stable two-column layout
- Left side: eyebrow, headline, subheadline, short paragraph, 2 CTA buttons, 3 info cards
- Right side: one strong hero product image only
- Below the hero, every category section should follow the same structure:
  1. category label
  2. heading
  3. short description
  4. one cover image
  5. three product cards
- Do not use awkward staggered layouts
- Use generous, consistent section spacing

## Typography rules
- Headlines should feel strong but not oversized
- Avoid very long paragraphs
- Use short supporting copy
- Keep card copy to one short sentence

## Product image rules
- Prefer clean white or light-gray backgrounds
- Center images
- Keep image display size visually consistent
- Do not stretch images
- Use only the strongest 3 SKUs per category on the main page
- Keep additional products for later expansion

## Safe change policy
- Make focused, reviewable edits
- Do not rewrite the website from scratch
- Do not introduce a framework unless explicitly requested
- Do not remove important business information
- Preserve:
  - company name
  - contact email
  - WhatsApp
  - OEM / ODM
  - lead times
  - MOQ

## Review workflow
Before finishing:
1. check visual hierarchy
2. check spacing consistency
3. check image paths
4. check mobile responsiveness
5. summarize changed files clearly

## What to avoid
- Overly busy backgrounds
- Too many competing accent colors
- Uneven card heights
- Long feature lists on the homepage
- Random layout experimentation
