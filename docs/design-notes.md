# RichLand design notes

## Desired visual feel
- calm
- premium
- modern
- balanced
- spacious
- trustworthy

## Avoid
- clutter
- awkward staggered layouts
- oversized paragraphs
- too many colors
- mismatched card sizes
- random visual experiments

## Homepage rules
- stable 2-column hero
- strong single product image on the right
- concise text on the left
- only 3 info cards:
  - 30 Years
  - OEM / ODM
  - 7 / 30 / 3 Days

## Category section rules
Each category section should use the same pattern:
1. category label
2. title
3. short description
4. one cover image
5. three product cards

## Product display rules
- keep image backgrounds visually clean
- keep product scaling consistent
- use equal card heights
- use generous whitespace
- prefer order over novelty
